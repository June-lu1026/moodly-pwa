# Moodly AI - UI & Scroll Fixed Version

修复内容：
- Chat 页面输入框不再和底部导航重叠
- Chat 输入框只在 Chat 页面出现
- App 内每个页面可独立滚动
- Insights 页面可以滑动到底部
- 保留动态记录、Calendar、Insights、Triggers、Localized Expression、语言切换
